About This Project

Project Overview

This web application uses advanced machine learning algorithms to classify text as either AI-generated or human-written. The system was trained on a comprehensive dataset of 58,537 text samples and achieves high accuracy in detecting AI-generated content.

With the increasing prevalence of AI text generation tools, this classifier helps identify synthetic content, ensuring transparency and authenticity in digital communications.

Key Features

  • Single Text Classification: Classify individual texts with detailed probability scores and confidence levels
  • Batch Processing: Analyze multiple texts simultaneously with comprehensive statistics
  • LIME Explanations: Understand which words influenced the prediction with feature importance analysis
  • Multiple Models: Choose between LightGBM and Logistic Regression based on your needs
  • Real-time Predictions: Instant classification results with probability distributions

How It Works

1. Text Preprocessing

Input text is processed using CountVectorizer and TF-IDF transformation to convert raw text into numerical features.

2. Feature Extraction

The system extracts unigrams and bigrams, removes stop words, and normalizes token frequencies for optimal classification.

3. Model Prediction

Pre-trained machine learning models analyze the features and generate probability scores for both classes.

4. LIME Explanation

LIME (Local Interpretable Model-agnostic Explanations) identifies the top words that contributed to the prediction.

Technical Stack

  • Backend: Flask (Python Web Framework)
  • Machine Learning: scikit-learn, LightGBM
  • Explainability: LIME (Local Interpretable Model-agnostic Explanations)
  • Frontend: HTML5, CSS3, Jinja2 Templates
  • Model Serialization: CloudPickle

Use Cases

  • Content authenticity verification for publishers and media organizations
  • Academic integrity monitoring in educational institutions
  • Quality control for content platforms and social media
  • Research on AI-generated text detection

Limitations

  • Models are trained on specific text domains and may not generalize to all content types
  • Performance may vary with evolving AI text generation techniques
  • Best suited for English language text
  • Short texts (< 50 words) may produce less reliable results
